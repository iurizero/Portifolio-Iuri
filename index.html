ShadowKey

Repositório

Este projeto foi desenvolvido com fins de ESTUDO. É um keylogger focado na captura de dados sensíveis do usuário, voltado para o estudo de segurança da informação na área de phishing. Ele captura dados como o login e senha do usuário e os salva numa lista de dados.

IntruSentry - Sistema de Detecção de Intrusão

Repositório

Um sistema de detecção de intrusão (IDS) desenvolvido em Python que monitora o tráfego de rede em tempo real, detectando atividades suspeitas e potenciais ameaças. O sistema utiliza análise de pacotes, detecção de anomalias e integração com logs para fornecer alertas em tempo real sobre possíveis ataques ou violações de segurança.
